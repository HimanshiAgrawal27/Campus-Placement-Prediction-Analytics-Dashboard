# 🎓 Campus Placement Prediction Dashboard

## 📌 Project Overview

This project is a Machine Learning based Placement Prediction Dashboard developed using Python and Streamlit. The system analyzes student academic and employability data to predict whether a student is likely to get placed during campus recruitment.

The dashboard also provides interactive visual analytics such as placement distribution, work experience analysis, gender-wise placement trends, and correlation heatmaps.

---

## 🚀 Features

- 📊 Interactive Streamlit Dashboard
- 🤖 Machine Learning Placement Prediction
- 📈 Placement Analytics & Visualization
- 📌 KPI Cards for Placement Statistics
- 🔥 Correlation Heatmap
- 💼 Work Experience Analysis
- 👨‍🎓 Gender-wise Placement Analysis
- 📁 Dataset Preview Section

## 📂 Dataset Information

The dataset contains student academic and employability details such as:

- SSC Percentage
- HSC Percentage
- Degree Percentage
- MBA Percentage
- Employability Test Score
- Work Experience
- Gender
- Placement Status

The target variable is:

- `status` → Placed / Not Placed

---

## 🤖 Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Model Training
5. Model Saving using Joblib
6. Streamlit Dashboard Deployment
7. Real-time Prediction

---
## 🎯 Model Performance

The machine learning model achieved approximately **86% prediction accuracy** on the campus placement dataset.

The model was trained using student academic performance, employability scores, and work experience features to predict placement outcomes.

## 📊 Dashboard Analytics

The dashboard includes:

- Placement Distribution Chart
- Degree Percentage Distribution
- Work Experience vs Placement
- Gender-wise Placement Analysis
- Correlation Heatmap

---

## 🎯 Prediction System

Users can enter student details from the sidebar, and the trained machine learning model predicts whether the student is likely to get placed.
